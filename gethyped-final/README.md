# GetHyped.nl Homepage Recreation

A professional recreation of the GetHyped.nl homepage built with React, Tailwind CSS, and Framer Motion.

## 🚀 Features

- ✅ Pixel-perfect design recreation
- ✅ Fully responsive (mobile-first approach)
- ✅ Smooth animations with Framer Motion
- ✅ Working hamburger menu
- ✅ Modern UI/UX
- ✅ Clean, maintainable code

## 🛠️ Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- React Icons

## 📦 Installation

\`\`\`bash
npm install
npm run dev
\`\`\`

## 🌐 Deployment

\`\`\`bash
npm run build
\`\`\`
